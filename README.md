# Ice Defense Simulator

Учебный проект по моделированию теплопроводности и таяния льда в замкнутой 2D-системе.

## Идея

Есть квадратная закрытая область. Внутри находится кусочек льда, источник тепла и несколько элементов, которые можно расставлять на поле. Элементы влияют на распространение тепла:

- `insulator` замедляет перенос тепла;
- `conductor` ускоряет перенос тепла;
- `cooler` отводит тепло из области.

Программа показывает, как распространяется температура и как меняется оставшаяся масса льда.

## Что полезного делает проект

Проект можно использовать как прототип игры или симулятора стратегии размещения защитных элементов. Пользователь меняет расположение объектов и смотрит, какой вариант лучше защищает лёд от таяния.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

В браузере откроется интерфейс. Нажмите `Run simulation`.

## Что показывается

- тепловая карта системы;
- положение льда, источника тепла и элементов;
- график оставшегося льда от времени;
- итоговая оценка защиты;
- риск полного таяния.

## Физическая модель

Используется упрощённое двумерное уравнение теплопроводности:

```text
dT/dt = alpha * Laplace(T) + heat_source - cooling
```

Лёд моделируется отдельно: пока температура ячейки льда ниже 0 °C, он нагревается. Когда температура достигает 0 °C, дополнительная энергия идёт на фазовый переход, то есть на увеличение доли растаявшего льда.

Модель демонстрационная, поэтому параметры выбраны так, чтобы процесс был наглядным и быстрым.
